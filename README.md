
# GG React Boilerplate

Ready-to-use boilerplate with configured environment


## Installation

Install with npm

```bash
  git clone https://github.com/GrzeCHu-a11y/react-boilerplate.git
  
  npm run install / or npm i

  npm run dev
```

## Or install by using "Use this template"
